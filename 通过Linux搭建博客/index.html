下载Xshell

通过Xshell链接服务器以及宝塔搭建

链接至Xshell
  • 打开软件–点击新建
  • 名称可随意填写、主机为服务器的公网IP地址
  • 点击左侧“用户身份验证”，用户名为root，密码为你所设置的密码
    搭建宝塔面板
  • 复制粘贴：
    yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh
  • 提示“是否搭建”，输入“y”后按回车
  • 等待直至出现下图即为搭建成功，记得记录好“账号密码”以及“进入面板的网址”

    打开云服务器的安全组

  • 通过服务器官网控制面板进入安全组
  • 为安全组的入规则添加规则，打开TCP端口：8888、888、80、20-21、22，来源填写0.0.0.0/0

    进入宝塔面板布置

  • 在浏览器上输入地址进入宝塔面板
  • 安装LAMP面板，PHP版本选择5.6，等待直至安装完成

    部署博客

  • 左侧栏选择“软件商店”，点击右边“一键部署”，找到WordPress并点击一键部署，并将自己的公网IP在“网站–添加站点”进行添加
